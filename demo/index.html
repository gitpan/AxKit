AxKit Demos

This page links to a set of demos for each technology in the AxKit toolchest. Please bear in mind that some of the demos may not work depending on what modules you happen to have installed. You can try each demo, and if it does not work, you will see an error page informing you of what you will need to install.

  • XSLT demos, using XML::LibXSLT
  • XSLT demos, using XML::Sablotron
  • XSP (eXtensible Server Pages) demos
  • XPathScript demos
  • AxPoint demos
  • XSL-FO demos
